# 游戏设计

> 历史状态：本文描述现有原型的旧设计。未来完整游戏设计已迁移到 `GAME_VISION.md` 及其列出的 17 份冻结文档；冲突处以后者为准。

## 核心原则

- 玩家控制人物而非国家；权限来自职位、组织、关系、财富、声望和情报。
- 世界即使在玩家职位较低时也自行运行。
- 人物的稳定性格与可变化政治倾向分离。
- 隐藏先天资质只产生温和修正，不作为技能硬上限，也不在正式 UI 显示数值。
- 常规长期行动可通过技能、准备、关系、组织支持和资源投入达到必然成功。
- 法理归属、军事控制和社会影响是不同数据，不以单一 `owner` 表示。
- 人物层级转换和继承保持身份连续性，不以同名新对象替代原人物。

## 信息呈现

正式 UI 显示玩家已知信息和定性成功把握，不显示 NPC 的完整隐藏倾向、未调查关系或精确成功百分比。开发模式可查看调试信息，但不得改变正常存档结构。

## 当前 Demo 状态

M7 已把地区社会影响与军事控制接入人物和组织行为：政策与社会活动改变地区影响但不移动前线，军事支援通过控制单元压力单独结算。人物可因死亡、退休、长期监禁、失势或自愿选择退出，并从真实关系或共同组织成员中选择继承者，在同一世界内部分继承资源、职位、情报、盟友和敌对关系。

M9 将正式行动目标接入人物、组织和控制单元：建立关系会创建稀疏联系，加入组织会登记成员和入口职位，争取职位会授予下一可用层级，调查会记录目标公开倾向，政策与控制支援进入各自地区通道。由此核心循环可以从行动面板完成，而不必依赖社会面板的直接调试按钮。

政府、军队、企业和工会通过职位提供权限，技能本身不能替代正式权力来源。实际接触才创建关系；120 名世界人物中只有玩家、组织领导和少数升级人物进入最多 20 人的活跃层，只有活跃 NPC 执行每日与月度简化 AI。

八类长期行动按权威游戏小时推进，可暂停、恢复、取消并因拘押或失能中断；技能、准备、支持、资金、权限、状态与阻力共同决定效率和结果。正式界面只显示定性成功把握，充分准备的常规行动可以达到必然成功。

玩家仍必须明确选择国家，可使用标准、全人口或按类别随机；相同条件与种子生成相同人物。正式人物信息只展示技能、已显现性格、已知倾向和当前状态，隐藏潜质与真实权重仅供显式开发检查。

地图填色表示实际军事控制，细边和地区边界表达法理/行政结构，斜纹表示争夺，橙色边表示相邻敌对控制者形成的前线；社会影响只在信息面板展示，不改变军事填色。

信息面板中的直接压力和易手按钮仍是 M3 机制验证入口，不代表玩家拥有国家级权限；正式行动和组织支援必须读取 M6 职位权限，并通过 M7 的社会或军事影响通道结算。

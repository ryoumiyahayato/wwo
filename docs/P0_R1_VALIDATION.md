# P0-R1 本机验证

网页端已提交 P0-R1 源码修复，但无法运行本机 Godot。以下步骤必须在 Windows 11 开发机执行，任何一步失败都不得标记 P0 完成。

## 1. 同步源码

```powershell
cd D:\wwo
git pull origin master
git status
```

预期：工作区干净，分支与 `origin/master` 一致。

## 2. 原有回归测试

```powershell
& 'D:\Tools\Godot-4.6.3\Godot_v4.6.3-stable_win64.exe' `
  --headless --path 'D:\wwo' `
  --script res://tests/test_runner.gd
```

验收：

- 退出码为 0。
- 不出现解析错误、脚本错误或阻塞警告。
- 历史检查全部通过；若测试数量改变，以实际输出为准，不得伪造。

## 3. 真实玩家旅程测试

```powershell
& 'D:\Tools\Godot-4.6.3\Godot_v4.6.3-stable_win64.exe' `
  --headless --path 'D:\wwo' `
  --script res://tests/p0_r1_player_journey.gd
```

该测试不得手工构造已完成行动，并要求正式控件同时满足可见、启用和位于视口内：

- 两次建立关系行动。
- 加入政府组织。
- 争取地区职位。
- 推动地区政策。
- 打开人物信息并返回，确认时钟、地图和行动连续。
- 退休并选择真实继承者。
- 使用正式保存按钮。
- 返回主菜单并加载，确认人物、时间和地区影响恢复。

验收：退出码为 0，末行显示 `P0-R1 PLAYER JOURNEY PASSED`。

## 4. 存档安全回归

```powershell
& 'D:\Tools\Godot-4.6.3\Godot_v4.6.3-stable_win64.exe' `
  --headless --path 'D:\wwo' `
  --script res://tests/p0_r1_safety_regression.gd
```

该测试使用隔离探针存档，不应覆盖正式手动档。它必须验证：

- 非法时钟状态导致恢复失败时，权威地图和时钟完全回滚。
- 失败恢复不会替换当前玩家会话。
- 不兼容配置版本在恢复前被拒绝。
- 主探针存档损坏时可以读取 `.bak` 安全备份。

验收：退出码为 0，末行显示 `P0-R1 SAFETY REGRESSION PASSED`。

  --headless --path 'D:\wwo' --quit-after 5

## 6. 1280×720 人工交互

```powershell
& 'D:\Tools\Godot-4.6.3\Godot_v4.6.3-stable_win64.exe' --path 'D:\wwo'
```

普通模式必须满足：

- 顶栏完整显示人物信息、长期行动、社会系统、保存游戏和返回主菜单。
- 长期行动面板列出 8 类行动。
- 不出现可任意填写的准备、资金、组织支持、关系支持或目标阻力输入框。
- 建立关系、加入组织和争取职位可以从长期行动面板选择真实目标。
- 社会面板不显示直接加入组织、指定职位、直接建立关系或人物升降级按钮。
- 地图侧栏不显示“施加敌方压力”和“模拟控制易手”。
- 人物信息页不显示隐藏潜质开关和宣传事件按钮。
- 保存游戏后按钮显示“已保存”，返回主菜单后“加载游戏”可用。
- 损坏或不兼容存档不会停留在无人物地图；界面应返回主菜单并显示错误。

## 7. 页面连续性

执行：

1. 推进至少 24 小时。
2. 开始一项尚未完成的长期行动。
3. 记录一个地区的社会影响或控制数据。
4. 打开人物信息。
5. 返回战略地图。

必须确认：

- 日期和小时未重置。
- 当前行动仍存在，进度未重置。
- 地图控制和社会影响未恢复初始值。
- 继续推进时间后行动继续增长。

## 8. AI 与世界自主变化

在 8 倍速运行至少 30 个游戏日，记录开始和结束数据：

- 至少一名活跃 NPC 的财富、疲劳、技能或组织状态变化。
- 至少一个地区社会影响变化，或至少一个边境控制单元的压力/争夺数据变化。
- 结算日志出现 NPC 日常行动或月度世界变化记录。

若 30 日没有任何可观察变化，P0-4 判定失败。

## 9. 保存加载

完成关系、组织、职位和地区行动后：

1. 点击保存游戏。
2. 完全退出程序。
3. 重新启动。
4. 点击加载游戏。

必须保持：

- 当前人物和继承历史。
- 日期、时间、暂停和速度。
- 财富、技能、声望和当前行动。
- 组织成员身份、职位和权限。
- 关系记录。
- 地区社会影响、军事控制、争夺度和前线。

## 10. 重新导出

所有源码和交互验证通过后再执行：

```powershell

  --headless --verbose --path 'D:\wwo' `
  --export-release 'Windows Desktop' `
  'D:\wwo\builds\windows\wwo-p0-r1.exe'
```

随后对导出 EXE 重复第 6 至第 9 节。Windows 10 仍保持“目标兼容，尚未实机验证”。

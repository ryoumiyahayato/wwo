# 世界、区域市场、战争与技术

状态：设计冻结 v1。本文冻结系统接口与因果顺序，不表示市场、战争、真实地图或技术树已经实现。

## 1. 总体依赖

```text
住户需求与劳动力
  ↕ 工资、消费、迁移
企业生产与现金流 ↔ 区域市场 ↔ 交通与国际贸易
  ↕ 税收、合同、监管         ↕ 知识、设备、人才
国家财政与制度执行 ↔ 技术形成、扩散与采用
  ↕ 外交利益、动员、征用     ↕ 军工、补给、通信
外交危机 → 战争 → 占领/破坏 → 市场、住户、制度与技术反馈
```

市场、战争和技术不是独立小游戏。每项结果必须回到人物、住户、企业、组织和机构的真实资源与状态。

## 2. 区域市场


基础商品类别：食品、衣物、住房、燃料、日用品、工具、机器设备、奢侈品。首版不继续拆成数十种微型商品。

每类商品保存生产、需求、库存、进口、出口、运输成本、成交量和价格。价格由供需缺口、可用库存、进口替代、运输和政策共同决定，并有数据驱动的单周期变化边界，避免瞬时爆炸。

## 3. 市场结算顺序

```text
企业提交产出与投入需求
→ 住户按实际生活水平提交消费需求
→ 政府与组织提交项目/军需需求
→ 现有库存与跨区运输形成可用供给
→ 按市场规则撮合聚合交易
→ 记录价格、成交、未满足需求和资金流
→ 企业利润、工资能力、住户满足度与税收更新
```

资金从买方账户流向卖方账户。进口和出口对应外国市场账户或贸易结算主体，不允许无来源生成财富。

## 4. 劳动市场与就业

区域劳动市场聚合职业、技能、工资、岗位和失业。企业根据产能、订单、工资成本与现金流创建或取消岗位；人物根据可达性、资格、工资、稳定性和住户需求求职。

就业合同是人物与企业间的正式连接。企业现金不足可能欠薪、减产、裁员、借款或破产；这些结果反馈到消费、压力、工会与政治。

## 6. 基础设施

铁路、港口、道路、河运和通信网络改变：

- 运输成本、容量与可靠性。
- 市场接入和企业利润。
- 通勤、旅行和迁移。
- 新闻与技术传播。
- 动员、军队移动和补给。

基础设施有位置、容量、维护、所有者/责任机构和损坏状态。没有下游效果的地图装饰不进入首版数据优先级。

## 7. 国际贸易

跨市场贸易读取价格差、运输成本、容量、关税、国境、战争、制裁和支付能力。首版使用聚合贸易订单，不逐商船模拟全部商品。

国际贸易同时传播企业关系、信息、人才和技术，并可能形成外交利益冲突。

## 8. 战争升级链

战争不能随机突然发生。正式状态链：

```text
利益冲突
→ 外交争端
→ 外交危机
→ 威胁、制裁、谈判或联盟活动
→ 军事准备
→ 动员
→ 宣战、退让或妥协
→ 战争
→ 停火/和谈
→ 和平条约与战后执行
```

冲突来源包括领土、市场、资源、债务、同盟、民族、殖民、政权安全和国内政治压力。每个阶段都提供人物、组织和国家在其权限内的反应机会。

## 11. 前线与作战

前线结果读取兵力、装备、训练、指挥、补给、基础设施、地形、士气、天气、战争支持、计划准备和多方向压力。任何一个攻击/防御总值都不能独立决定全部结果。

作战按战区周期和事件边界结算，不为每名普通士兵创建逐小时战术节点。控制变化产生原因记录并反馈人口、市场、机构和通信。

## 12. 不同身份的战争体验

- 平民：配给、物价、军需工作、征兵、伤亡消息、迁移和政治活动。
- 边境居民：撤离、财产损失、短缺、占领、合作或抵抗。
- 企业人物：合同、扩产、原料、运输、征用和战争利润/破产风险。
- 普通军人：训练、单位关系、调动、任务、伤病和晋升。
- 军官：按指挥层级获得计划、调动与补给请求权限。
- 政府人物：按职位参与动员、军费、战争目标、外交和和谈。

普通人物能观察战争并承受结果，但不能直接控制全国军队。

## 13. 技术三阶段

### 知识形成

大学、研究机构、企业实践、军队、发明者和外国知识产生研究积累。

### 知识扩散

受距离、贸易、教育、语言文化、人员流动、开放程度、专利、保密、间谍和许可影响。

### 实际采用

拥有知识不等于已经使用。采用需要资金、设备、工业条件、人才、基础设施、组织意愿和政策。

## 14. 被动与定向研究

满足前置条件后，国内实践、外国扩散和科研积累会产生被动进度，避免关键技术被一次随机事件永久锁死。

定向研究需要现代科研机构、预算、管理制度和专业人才。投入有边际递减；职位人物只能为自己有权限的国家、企业、大学或军队选择方向。

技术可通过许可、购买、专家、留学、企业合作、战争缴获和间谍引进。特殊事件可加速或发现应用，但不能成为永久随机门槛。

## 15. 市场—战争—技术闭环

- 市场为军队提供税基、军费、装备、食品和运输。
- 战争破坏生产、运输和人口，改变价格、就业、债务和国家合法性。
- 技术提高生产、通信、医疗和军事能力，但需要市场设备与制度采用。
- 国家政策影响教育、研究、关税、动员和基础设施，但受财政与执行能力限制。
- 人物通过工作、企业、组织、职位和军事服务进入闭环，而非直接拥有系统。

## 17. 首版与保留接口

市场首版必须：8 类商品、区域价格/库存/运输、住户消费、企业现金流、就业和破产基础链。只保留全球金融、数十商品、逐运输工具和复杂汇率接口。

战争首版必须：升级链、和平/战争状态、动员、陆军战区/前线、补给、人物战争体验和权限。海军、空军、复杂外交条约和战术战斗先保留接口。

技术首版必须：因果前置、被动扩散、机构定向研究、组织/地区采用和引进。完整全球科技内容、专利市场和间谍科技网先保留接口。

# 《1900》世界权威说明

## 当前正式世界

当前正式地图与世界地理来自：

```text
data/world_map/
scripts/world_map/
```

其中包含离线国家轮廓、国家目录、法国宏观地区与行政几何、城市、港口、铁路、公路、航线、机构、组织、地图模式和几何缓存。

当前几何主要来自现代 Natural Earth，作为1900年正式历史边界完成前的过渡数据。国家主题名称和法国详细地区用于验证世界地图、缩放、LOD、对象交互与多精度模拟，不代表所有历史疆域已经准确完成。

## 正式方向

《1900》的正式世界采用真实世界地理与多精度模拟：

- 玩家当前地区：详细人物、地点、日程、旅行、工作、企业和组织；
- 重要关联地区：简化工资、价格、企业、组织和事件；
- 远方地区：聚合人口、经济、政策、贸易和战争状态。

所有层级必须引用同一国家、地区、城市、地点和交通 ID 体系。

## 已撤销的架空世界

`world:loran_vesta_1900`、洛岚联邦、维斯塔联盟、八个架空地区、四个模板城市和80个正交网格单元不再是正式世界。

相关内容保留在Git历史和 `data/alpha/world.json` 中，只能用于：

- 历史测试；
- 性能夹具；
- 存档迁移研究；
- 经济、合同、企业和政治服务的待迁移验证。

普通启动、正式存档和发布文档不得把这些内容描述为正式 Alpha 世界。

## 现有系统迁移原则

时间、日程、地点、旅行、关系、知识、消息、人物、账本、资产、合同、债务、企业、组织、职位、政策、AI和存档代码应尽量保留，但必须改为读取正式地图的国家、地区、城市、地点和交通引用。

不得因为旧网格具有稳定ID、自动测试或较低性能成本，就重新把它升格为产品世界。

## 当前发布状态

普通入口已恢复到V2.3正式世界地图。旧架空Alpha处于待迁移和待审查状态，不能作为完整可玩Alpha交付。

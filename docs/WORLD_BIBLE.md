# 《1900》Alpha 世界圣经

## 1. 世界范围

`world:loran_vesta_1900` 是首个完整 Alpha 的有限正式世界。时间从
1900-03-12 05:00 开始；洛岚联邦和维斯塔联盟已经存在并持续运行，玩家只是其中一名
具有既往履历的成年人。两国共有八个行政地区、80 个基础地图单元、四座主要城市和
32 个可进入地点。世界不围绕玩家生成，也不会因玩家无操作而暂停经济、职业、债务、
组织和政策变化。

本世界只使用架空国名。历史技术、法律和社会能力保持 1900 年前后可理解的范围：
铁路、邮政、机械工业、地区银行、商事法院、工会、政党社团和报纸已经存在；复杂证券
清算、电子通信、现代福利国家和战术战争指挥不在 Alpha 范围。

## 2. 地理与交通

核心地形是 10×8 的统一拓扑。每个单元保存国家、地区、土地用途、人口、主要阶层、
产业、工资、生活成本、资源条件、交通、基础设施、治安/政治环境、组织影响和当前经济
状态。每个单元至少影响就业、工资、价格、经营地点、运输、信用、需求、政治支持、
组织活动、治安或行政权限之一。

四座主要城市：

- 晨港：洛岚首都与港口，贸易、公共行政、航运和报业集中。
- 铁川：洛岚机械制造、河运和专业服务中心。
- 星城：维斯塔首都、金融、纺织、联盟行政与政治社团中心。
- 赤原：维斯塔重工、煤炭、军需和高风险企业接管中心。

每座城市使用相同八类地点：住宅、工作场所、市场、信贷与商事所、组织会所、行政公署、
铁路与货运站、公共广场与邮局。地点的业务能力来自服务标签；城市没有专用业务函数。
四城内部由步行和城市交通连接，城际由铁路连接。正式图包含 34 条稳定边，其中至少两条
跨国连接。

## 3. 人物与社会

Alpha 初始生成 120 名背景人物，并提供 8 个玩家预设。人物同时包含出身和经历、隐藏
资质、显性技能、行业/组织/地区经验、资格、性格、倾向、关系、知识、健康与当前状态、
财富、债务、资产、职位和组织身份。

隐藏资质为学习、推理、社会感知、意志与自控、身体协调和压力承受。普通 UI 只显示
可观察表现，不显示精确值；开发模式可查看真相。技能和经验改善判断、方法、成果质量、
工期、返工、超支和对专业协助的依赖，但不会删除工作、借款、创业、买卖、签约、加入
组织或申请职位的入口。

活跃高精度人物上限为 20。玩家、当前接触人物、重要合作者、对手、代理人和直接关联
事项对象可升级；其余人物按日或周聚合。降级保留稳定 ID、地区、职业、财富/债务区间、
资产、合同、关系摘要、知识、职位、组织身份、健康和重要事件。

人物倾向由安全、财富、地位、归属、理念五种驱动力，以及谨慎—冒险、协商—强制、
守规—机会主义三组方式倾向组成。人物还保存议题立场、最多少量主要议程和底线。
倾向影响 AI 偏好，不禁止玩家选择。

## 4. 经济循环

经济使用整数最小货币单位和统一账本。正式商品与服务为粮食、生活用品、煤炭、木材、
机械、纺织品、运输服务、调查与信息服务。地区供需、基础设施、运输、战争和政策共同
改变价格。背景居民需求聚合结算，玩家与企业交易保留正式订单、合同和账本来源。

企业不是独立小游戏。零售/地区贸易、小型生产、物流服务和报纸/调查服务共用组织、
资产、合同、职位、人员、订单、账本和市场。企业可以创建、购买、出售、合伙、注资、
借债、雇佣、外包、接单、采购、生产/服务、运输、定价、销售、支付工资、扩张、收缩、
违约、破产和解散。

稳定生活以自动预算运行，包括必要支出、居住、交通、债务支出、发展支出、资产收入、
企业收入和税费。现金不足、价格显著变化、居住/健康变化、无法偿债或主动调整生活水平
才进入前台。

## 5. 资产、合同与债务

资产包括现金账户、债权、企业权益、房产、土地、设备、库存、运输工具、抵押权和有价值
合同。所有权与实际控制权分别保存。资产可获得、出售、转让、继承、抵押、租赁、没收、
损坏、贬值和在破产中处置。

七类合同为雇佣、买卖、服务、租赁、借款、合伙和订单。每份合同保存参与方、标的、金额、
期限、义务、付款和交付条件、违约条件、担保、履行状态、证据与文件。合同可正常履行、
延期、部分履行、重谈、违约、欺诈、争议，并通过正式执行或私下和解结束。

债务不是平行余额：它是借款合同的未履行本金和费用、债权人的债权资产、抵押引用以及
相应账本交易。信贷支持无抵押个人借款、有担保个人借款、企业经营借款、资产抵押借款和
供应商信用，并覆盖申请、审查、报价、接受/拒绝、放款、费用、还款、提前偿还、逾期、
违约、抵押处置、担保、再融资、借新还旧、重组、债权转让和破产清算。

## 6. 组织与政治

企业、政府、工会、政治社团、报社/公共组织和军队均使用统一组织对象。组织保存成员、
职位、资产、账本、合同、方针、事项、对外关系、辖区、公开立场和内部派别。

职位是权限包，包含辖区、信息访问、预算、可命令人员、任命/罢免权、签约权、政策权限、
责任、任期和取得/失去条件。职位可加入组织后争取、任命或表决获得，也可辞去、被罢免、
任期结束、随组织变化或因违法丧失。

政策实施依次结算法定权限、实际服从、执行能力、资金人员、基础设施和外部阻力。成功
实施才修改地区工资、就业、企业成本、信用、运输或其他正式状态。反对来自现有利益、
资源和制度冲突，不每天凭空生成敌人。

腐败可转移合同、接受利益、挪用预算、保护盟友、低价取得资产或使用公共人员处理私人
事务。每次行为必须留下受益者、资金/资产流、文件、知情者、关系义务和证据。调查可能
发现、被掩盖或证据不足，并产生政治、法律、关系、职位和资产后果。

## 7. 信息边界

世界真实状态、人物已知事实、人物估计、未经确认的信息、公开信息和开发真相分别保存。
人物对现金维持期、债务成本、企业状况和政策前景的估计根据推理、财务、行业经验和信息
质量形成区间；高能力人物通常更准确，但不会永久完全正确。

重大结果先产生世界事实和来源，再改变数字。正式事件至少说明行动者、原因、对象、实际
发生内容、即时结果、长期影响和是否需要决定。界面可显示期限、风险与估计，不推荐路线，
不生成任务清单，也不以技能隐藏整个系统。

## 8. 战争与技术骨架

两国初始和平。战争状态可提高运输成本、缩减信用、改变就业、贸易和政治预算；军队是
统一组织，军事职位、征集和预算按聚合规则结算。生产方法可读取工程技能、设备和专业
经验。本 Alpha 不实现战术单位控制、完整战役、科技树或详细武器模拟。

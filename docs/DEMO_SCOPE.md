# Demo 范围

> 历史状态：这是旧 M0～M9 Demo 范围，不再是未来产品范围或实施路线。新设计以 `GAME_VISION.md` 与 `DEVELOPMENT_ROADMAP_REDESIGN.md` 为权威；当前项目仍未达到 Demo。

## 必做

- 两个架空国家、八个行政地区、约 64 至 96 个控制单元与动态前线。
- 玩家明确选择国家和随机模式，获得一名可持续成长的社会人物。
- 权威小时制时钟、暂停及 1/2/4/8 倍速，模拟与渲染帧分离。
- 人口群体、背景人物和最多 20 名活跃人物的分层模拟。
- 八类基础行动、组织与职位、按需关系、简化 AI、人物退出与继承。
- 固定随机种子、安全存档、基础开发工具和 Windows x86-64 导出。

## 可选

- 在不延误必做项时改善占位图形、UI 动效和调试可视化。
- 为 Linux 与 macOS 准备未声明为已验证的导出配置。

## 明确不做

完整经济市场、全球贸易和外交、完整选举法律系统、即时战术战斗、空军海军、真实历史人物与地图、家庭遗传婚姻、自由行走、多人、模组、成就、Steam、云存档、配音、正式美术和在线生成式 AI。

## Demo 验收标准

玩家能完成“选择国家与人物 → 推进时间 → 执行动作 → 加入组织并争取职位 → 影响地区 → 观察控制变化 → 人物退出并继承 → 保存加载继续”的闭环。测试必须覆盖确定性、地图边界、继承、无效存档和长期模拟，且文档与实际实现一致。

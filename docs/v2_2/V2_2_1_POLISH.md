# V2.2.1 界面与地图交互收尾

本轮根据 V2.2 实机评审反馈完成收尾，不扩展 V2.3 系统范围。

## V2.2 专用入口

V2.2 不再返回旧 P0-R1 主菜单，也不再从旧菜单进入旧网格地图。

- 项目默认入口改为 `res://scenes/v2_2/v2_2_life_loop_menu.tscn`。
- V2.2 齿轮菜单中的返回项改为“返回原型菜单”。
- “开始新的 V2.2 评审”只进入 `res://scenes/v2_2/v2_2_life_loop_main.tscn`。
- “载入 V2.2 最近存档”只读取 `v2_2_life_loop_1` 评审存档，再进入同一 V2.2 世界地图。
- 旧 P0-R1 菜单和旧网格地图仍保留在仓库中用于历史回归，但不再属于 V2.2 用户路径。

## 齿轮菜单

- 删除无必要的“系统工具”标题。
- 菜单只显示实际操作项。
- 菜单宽度保持 244 像素，所有项目在 1280×720 下应完整显示。

## 联系人和人物面板

联系人不再由界面写死为让娜。

- 联系候选来自当前人物的正式关系记录。
- 候选名称读取 `target_display_name_zh`，缺失时依次回退到原文名和稳定 ID。
- 皮埃尔当前可联系让娜·勒鲁瓦。
- 阿尔贝没有对应关系记录，因此不显示让娜，也不能安排该联系。
- 联系活动、人物关系页、通知、当前活动和下一活动使用同一关系数据源。
- 无联系人时显示“当前人物没有可用的关系行动”。
- 关系人物标题与首个人名之间增加固定间距，避免中文姓名与标题重叠。
- 概览中的“当前状态”标题上移，和状态卡片形成明确间隔。

## 安排活动面板

- 打开面板后权威时间暂停。
- 关闭面板后恢复打开前的暂停状态和倍率。
- 暂停变化立即刷新时间显示。
- 日期、开始、结束和持续时长集中到一个带边框的时间编辑区。
- 调整按钮缩短为“日− / 日+ / 时− / 时+ / 时长− / 时长+”。
- 联系活动表单保存真实 `related_entity_id` 和显示名称。

## 地图边缘滚动

V2.2 主场景保留鼠标边缘滚动：

- 有效边缘宽度：20 像素。
- 平移速度按鼠标接近边缘的程度在 240—720 像素/秒之间变化。
- 右边缘向右移动镜头，左边缘向左移动，上下边缘同理。
- 位于 UI 控件上、按下鼠标或正在拖动地图时不触发。
- 边缘滚动复用地图已有 `begin_camera_interaction` / `pan_by` / `end_camera_interaction`，不会重建地图几何。

- 项目默认入口为 V2.2 专用菜单；

- 返回项不指向旧 P0-R1 菜单；
- 联系人首行与标题保持安全间距；
- “当前状态”标题使用上移后的偏移；
- 原有联系人、暂停恢复和边缘滚动回归继续保留。

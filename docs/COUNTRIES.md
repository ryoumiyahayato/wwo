# Alpha 国家设定

## 洛岚联邦（`country:loran_federation`）

首都晨港。人口 462 万，通行洛岚通用语，货币为洛岚克朗。联邦由沿海商港、河谷工业区
和南部农业共同体在六十年前缔结，现为议会联邦制。联邦议长是国家元首，议会任命行政院；
地区议事会负责地方预算和许可，地方事务厅承担执行。

地区议员经选举产生，行政官由议会任命。职位因辞职、罢免、任期届满、组织变化、违法或
失去支持而结束。职业文官、地区法院和城市警务局分立；偏远地区服从与执行能力弱于晨湾。
军队受联邦预算与文官国防委员会约束。

登记财产和抵押权受法院保护。地区商事处登记企业、合伙人和实际控制人；商事法院可判令
履行、赔偿、重组或抵押处置。银行、商人信用和私人担保借贷并存。财政来自营业税、工资税
和港口关税。工会合法，地区劳动委员会监督集体谈判。破产先尝试重组，失败后清算；担保
债权优先，工资债权受有限保护。

产业为港口贸易、机械、纺织、农业加工与报业。进口煤炭和木材，出口机械、纺织品、调查
与航运服务。主要阶层为产业工人、商人、文官、专业人士和农民。晨港与铁川工资高但住房
贵，南岭生活成本低但信用和公共服务不足。公共学院、行业学徒、工会、文官考试和经商构成
社会流动渠道。晨湾掌握贸易和财政，铸原承担工业风险，南岭长期要求更多公共投入。

主要组织包括联邦行政院、商事登记处、劳动委员会、联邦陆军、企业与商贸组织、两类工会、
公民同盟、劳动议政会、公共学院和报社。非法私人放贷与企业—官员利益网络存在，但不是
公开合法组织。

持续国家问题：

1. 港口关税由联邦还是地区分配。
2. 工厂安全、工时和地区工资底线。
3. 南岭交通、教育和信贷长期不足。

可争议议题为最低工资、企业许可、信贷费用和南岭铁路。可实施政策为地区工资底线、快速
商事许可、信贷费用上限和南岭铁路扩建。预算领域为劳动与健康、商业与信用、运输与教育。

## 维斯塔联盟（`country:vesta_union`）

首都星城。人口 431 万，通行维斯塔联盟语，货币为维斯塔塔勒。矿区、湖区与边地城市在
王朝解体后组成联盟，由地区代表和登记产业团体共同议政。联盟执政委员长是国家元首；
议政厅选举执政委员会，行政署按地区执行预算和政策。

地区代表会、行政署与行业公所分享地方权力。代表由地区和登记社团推举，行政职位由委员会
任命；任期、内部表决、纪律调查和违法可导致失位。行政署、联盟法庭和地方卫戍警备分工，
行业公所影响执行。卫戍军向委员会负责，但军官团保留较强内部自治。

城市财产登记完善，边地土地和运输权仍常依赖习惯合同。联盟商会登记企业，并要求披露主要
合伙人和债务。合同可由联盟法庭、行业仲裁或私人和解执行。产业银行、供应商信用、合作社
信贷和私人放贷并存。财政来自生产税、牌照费、铁路费和边境关税。工会可登记，罢工和仲裁
规则随地区变化。破产由债权人会议先议定重组，失败后拍卖资产并终止组织。

产业为重工、煤炭、湖运、地区贸易和公共报业。进口粮食与纺织品，出口煤炭、机械和运输
服务。主要阶层为矿工、产业雇员、地方商人、官员和军官。星城信用充足但生活昂贵，赤原
工资和就业随重工订单波动，北辰经营成本低但边境风险较高。联合学堂、军官学校、技术班、
合作社和地区政治提供社会流动。银野掌握金融，赤丘承担重工债务，东湖依赖运输投资。

主要组织包括联盟议政厅、商事署、铁路委员会、卫戍军、产业和商业组织、工会、改进社、
产业团、联合学堂和日报社。矿区私人保安、非法放贷人与地方政治网络构成灰色权力。

持续国家问题：

1. 赤丘重工债务、订单波动与失业。
2. 星城、东湖和赤原之间的铁路预算分配。
3. 文官委员会与军官团的权限边界。

可争议议题为连续工时、工厂牌照、企业债务重组和东湖运输预算。可实施政策为工时限制、
工厂检查、企业债务协调处和东湖运输拨款。预算领域为劳动与检查、产业与信用、铁路与
公共服务。

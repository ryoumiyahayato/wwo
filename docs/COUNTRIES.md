# 国家数据状态

## 正式国家目录

当前正式国家目录位于：

```text
data/world_map/countries.json
```

该目录包含177个国家或政治实体记录，并由 `data/world_map/world_coastlines.json` 和 `data/world_map/map_geometry_cache.json` 提供地图几何与LOD缓存。

部分主要国家使用1900年主题名称，例如法兰西第三共和国、大清帝国、俄罗斯帝国、德意志帝国、奥匈帝国、奥斯曼帝国、日本帝国和大英帝国。当前边界几何主要来自现代Natural Earth，是后续历史边界完善前的过渡权威。

## 内容限制

- 国家主题名称不等于完整历史疆域已经准确。
- 国家制度、经济、人口、组织和政策内容仍需逐步接入同一国家ID。
- 普通UI应区分公开事实、人物已知信息和开发真值。
- 远方国家可以使用聚合模拟，但不得从正式目录中消失或被架空国家替代。

## 已撤销内容

洛岚联邦和维斯塔联盟不再是正式国家。

以下ID只允许出现在旧架空Alpha夹具、迁移测试和历史回归中：

```text
country:loran_federation
country:vesta_union
```

普通启动、正式存档、正式地图、国家说明和后续Alpha不得继续以这两个国家作为产品世界基础。

## 后续接入要求

经济、劳动、企业、合同、债务、组织、职位、政策和AI系统必须引用 `data/world_map/countries.json` 中的稳定国家ID。需要详细制度数据时，应按这些ID扩展内容，而不是创建与地图目录平行的第二套国家。

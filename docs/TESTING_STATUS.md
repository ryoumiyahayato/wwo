# 当前测试状态说明

## 自动测试可以证明

- 正式组合根可初始化；
- 现有权威服务没有被社会沙盒复制；
- 玩家与NPC进入统一任务结构；
- 日程、地点和旅行参与任务；
- 事件顺序、逐行动回滚、认知隔离、存档和确定性在当前覆盖范围内成立；
- 玩家零输入30日和一年运行保持状态上限与性能预算；
- Windows可执行文件和安装包能够进入CI构建链。

## 自动测试尚不能证明

- 所有现实可行行动均已实现；
- 每种方法都有完整而独立的社会后果；
- NPC行为已经具有足够人物差异和长期人生变化；
- 地图和计划界面已经有足够玩法价值；
- 普通玩家能够清楚理解所有因果；
- 完整多阶段即时反应、多原因事件和错误认知网络已经完成。

## 人工测试只负责判断

- 世界是否会自行变化；
- NPC是否像具体人物；
- 玩家是否能自由选择方向；
- 日常手续是否过多；
- 因果是否清楚；
- 地图和计划是否有用；
- 不同方法是否产生可感知的不同后果；
- 玩家是否愿意继续观察和介入。

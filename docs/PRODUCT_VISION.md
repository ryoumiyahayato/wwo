# 产品愿景

> 已由 `GAME_VISION.md` 取代为完整产品愿景。本文仅保留早期愿景记录。

## 长期愿景

构建一款从 1900 年开始的二维离线社会与战争模拟游戏。玩家不是国家意志，而是一名受到职业、职位、财富、技能、组织与人际关系约束的社会人物。世界在玩家影响力较低时仍会按独立规则运转，人物退出后可从真实关系网络中选择继承者。

长期体验强调可解释的社会流动、人物连续性、地区社会影响与军事控制的区分，以及相同输入可复现的模拟结果。

## 当前边界

长期愿景不等于当前 Demo 范围。首版仅验证 `docs/DEMO_SCOPE.md` 中的核心循环，不包含完整经济、外交、法律、战术战斗、真实历史、多人或在线服务。开发必须按 `docs/ROADMAP.md` 逐里程碑推进。

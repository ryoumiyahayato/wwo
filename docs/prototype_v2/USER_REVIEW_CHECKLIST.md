# V2 静态原型用户检查清单

请在 1280×720 独立场景中人工检查：

- [ ] 整体是否仍像后台管理系统。
- [ ] 地图是否真正占满整个窗口，而非中央小框或背景板。
- [ ] 四角位置、面积与语义是否合理。
- [ ] 人物、组织、国家和地图对象是否清晰分离。
- [ ] 普通工人与地方官员看到的信息和动作是否明显不同。
- [ ] 普通人物是否仍有完整、易懂的个人界面。
- [ ] 关系人物卡是否自然，是否避免全人物名单和精确陌生人数据。
- [ ] 已有关系能否从卡片直接联系。
- [ ] 我的组织与探索组织是否真正分开。
- [ ] 已加入组织是否显示具体职位、部门、上级、权限和下一职位。
- [ ] 信息是否过密，扫视、悬停和点击详情层级是否合理。
- [ ] 上下文行动是否比固定八按钮更符合对象语义。
- [ ] 左键、再次点击收起、地图拖动、滚轮缩放和 Esc 是否符合直觉。
- [ ] 和平模式是否没有前线，战争模式是否明显只是静态视觉示例。
- [ ] 世界动态是否分级并聚合重复消息。
- [ ] 是否存在大段说明书文字常驻地图。
- [ ] 是否存在 Excel 式单行表格、大型下拉框或顶部按钮长条。
- [ ] 是否存在点击穿透、面板叠加冲突或遮挡后仍可点击。
- [ ] 是否存在内容伸出屏幕、文字重叠、截断或裁切。
- [ ] 是否愿意批准该视觉与交互方向进入下一阶段。

截图和自动测试不是用户批准。只有用户实际检查并明确同意后，才能进入住户、时间预算、生活经济、有限认知、企业现金流、区域市场、真实地图、政治、战争、技术或社会继承的正式实现。

# V2.1 地图与信息架构视觉修订记录

## 修订原因

V2 第一版技术骨架可用，但用户视觉验收未通过：地图像拓扑拼图，地理层级混排；左侧大面板压住地图；身份差异主要是字段增删；对象动作和地图模式仍像后台按钮阵列；世界动态仍是常驻文本框。

V2.1 没有增加正式功能，只重构地图视觉和信息架构。

## 保留的技术骨架

- 独立场景和 `prototype_only: true` 静态数据隔离。
- 地图拖动、滚轮缩放、对象选择和高亮。
- Esc 逐层关闭、主面板互斥和短动画。
- 普通工人 / 地方官员评审身份。
- 有限关系人物、我的组织 / 探索组织分集、上下文行动和状态符号。
- 原型 smoke 与旧项目统一验证框架。

## 已替换的视觉结构

- Natural Earth 世界国界/海岸线替代任意多边形拼图。
- 经纬度和统一 Robinson 近似投影替代屏幕像素坐标。
- 世界 / 国家地区 / 地方三档语义缩放替代等比放大文字。
- 396 px 地图优先侧栏替代接近半屏的大管理面板。
- 纵向文字导航、对象入口和有限上下文动作替代页签/按钮矩阵。
- 一条摘要式世界动态和展开历史替代常驻动态文本框。
- 单个地图模式入口替代四个常驻宽按钮。
- 普通工人公开门户与地方官员机构层级图替代同页字段增删。

## 地图数据与边界声明

世界轮廓来自 Natural Earth Vector 1:110m Admin 0 Countries。Natural Earth 官网声明其矢量和栅格数据为公共领域。生成工具 `tools/prototype_v2/build_world_geometry.py` 下载官方维护仓库的 GeoJSON，保留既有 1:110m 简化，剥离无关属性并离线输出原型 JSON。

当前世界政治边界采用现代 Natural Earth 几何，仅用于 UI、缩放、标签和地图模式验证。国家名称及颜色包含 1900 年主题占位，不代表准确的 1900 年疆域、殖民体系或国际关系。

## V2.1.1 内容一致性修订

- 左上国家代码改为自制蓝白红圆形三色徽记，国家详情显示“法兰西第三共和国”和 `République française`；稳定 ID 与数据简码不进入正式 UI。
- 默认人物、关系人物、企业、工会、夜校、合作社、机构和动态统一到法国里尔—北部省背景，并继续标记为静态原型。
- 游戏宏观地区使用地理经济名称；北部省、里尔区和里尔市使用独立行政单位类型与稳定 ID。
- 欧洲中景不再显示法国内部宏观地区；法国近景遵守地区、城市和铁路预算，机构/组织节点只在选择城市、地区或对应对象后显示。
- 本次修订没有制作历史疆域、正式制度模拟、市场结算或政策运行逻辑。

## 当前结论

- V2.1 仍是静态视觉与交互原型。
- 未接入正式模拟、服务、存档或正式数据。
- 自动测试只能验证结构性约束。
- 截图和本地点击检查不能代替用户视觉判断。
- 尚未获得用户批准，不得进入正式业务接入阶段。

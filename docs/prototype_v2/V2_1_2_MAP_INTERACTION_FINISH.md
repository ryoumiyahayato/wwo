# V2.1.2 地图与交互收尾说明

## 地图缺块诊断与修复

世界几何生成器现在遍历 Natural Earth Admin 0 的每个 `Polygon` 与 `MultiPolygon` 外环，并保留真实内环。对象稳定 ID 优先使用 `NE_ID` 和行政代码；`ISO_A3=-99` 时回退到 `ADM0_A3` 等稳定字段，仍无法识别时才生成确定性哈希 ID。缺少专题颜色的陆地统一使用中性陆地色，禁止透明或海洋底色。

原缺块对象是苏丹。上游环在 `[33.9634, 9.4643]` 含自相交尖刺，Robinson 近似投影后的闭合环又在 `[23.887, 8.6197]` 暴露第二处尖刺，导致 Godot 多边形三角化丢弃整块陆地。生成器现以通用自相交检测删除退化尖刺，并把对象、坐标、阶段和修复动作写入 `prototype_world_coastlines.json` 的审计记录；没有手绘补丁多边形。

## 法国两层地理结构

行政层来自 Natural Earth 1:10m Admin 1 法国本土几何，共 96 个自然多边形单位。它们具备稳定 ID、行政层级、父国、几何、标签锚点、标签优先级、可见缩放和中法名称。里尔区与里尔市保留小范围不规则评审占位，作为北部省下级语义节点。

九个游戏宏观地区只保存行政单位稳定 ID 列表，运行时通过 `Geometry2D.merge_polygons` 合并，不再自行绘制矩形。宏观覆盖使用半透明填充与粗金色外边界，行政单位使用细浅色线，近景顺序为国家、宏观覆盖、行政边界、城市与交通。

这些边界是可靠的现代简化占位，不等同于完整 1900 年法国历史行政地图；历史塞纳省等制度边界仍待后续专门录入。

## 标签、镜头与交互范围

- 世界国家数据覆盖 177/177 个要素，全部提供中文显示名、正式中文名、原名、标签优先级、最小可见缩放与标签锚点。
- 世界远景只常驻主要国家、当前国家、关联/选择国家和悬停国家；欧洲中景按预算与碰撞逐步增加中小国家。
- 最大缩放由 12 提高到 96；法国聚焦使用 18，里尔/北部省聚焦使用 96。
- 双击左下人物入口聚焦里尔，双击左上国家入口聚焦法国，`Home` 返回世界视角。
- 平移边界按缩放后的地图包围盒约束，世界不能被完全拖离视口。

## 静态原型边界

本轮时间速度、个人经济、计划效果、关系行为、组织职位和系统菜单均为静态交互与信息架构原型。没有接入正式时间推进、经济结算、政策执行、市场、战争、AI 或存档改造。

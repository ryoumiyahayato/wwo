# 《1900》V2.1.1 文化与地图信息原型范围

## 当前状态

V2 第一版工程实现与自动交互验证完成，但用户视觉验收未通过。第一版的原型隔离、输入、面板互斥、Esc 层级、静态身份与测试骨架继续保留；抽象拼图地图、任意屏幕坐标、大侧栏、重复人物摘要、常驻身份切换器、按钮阵列和常驻动态框已在 V2.1 重构。

V2.1.1 在保留 V2.1 地图骨架和四角信息架构的前提下，只修订法国文化一致性、制度命名、行政层级语义和西欧/法国标签密度。它仍是静态视觉原型，未接入任何正式业务系统，也尚未获得用户批准。

## 明确不包含

- 不实现住户系统、时间预算、个人生活经济、企业现金流或区域市场结算。
- 不实现正式政治、战争、技术、历史世界地图或新存档格式。
- 不接入人物行动、关系、组织、AI、市场、战争、技术、时间或存档服务。
- 不读取或写入 `user://`；保存、设置、自动暂停和返回只展示视觉分组。
- 不修改正式 `main_scene`，不改变正式游戏状态。

## 地图数据范围与许可

完整世界海岸线和现代国家轮廓来自 Natural Earth Vector 1:110m Admin 0 Countries：

- 上游文件：`ne_110m_admin_0_countries.geojson`
- 上游仓库：<https://github.com/nvkelso/natural-earth-vector>
- 官方条款：<https://www.naturalearthdata.com/about/terms-of-use/>
- 许可：公共领域
- 处理：保留 Natural Earth 自身 1:110m 简化；只保留外环和必要属性；坐标保留四位小数；未再进行会改变拓扑的简化
- 用途：本轮仅用于离线视觉原型

当前世界政治边界采用现代 Natural Earth 几何，仅用于 UI、缩放、标签和地图模式验证。国家名称及颜色包含 1900 年主题占位，不代表准确的 1900 年疆域、殖民体系或国际关系。

法国与西欧是唯一细化示例：9 个游戏宏观地区、11 个法国重点城市、港口、铁路、公路、航运和少量机构/组织节点。北部省、里尔区和里尔市是独立的行政语义链；游戏宏观地区不等同于历史行政区划。其他世界区域保持粗粒度。

俄罗斯帝国、大英帝国、奥匈帝国、奥斯曼帝国和非洲殖民体系的准确历史疆域均未完成；界面中的相关名称只是 1900 年主题占位。

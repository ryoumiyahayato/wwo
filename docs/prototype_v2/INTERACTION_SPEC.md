# V2 静态原型交互规范

## 四角交互

- 左上：国家与当前可接触机构；点击展开国家与机构中心。
- 右上：紧凑日期、暂停、速度与自动暂停摘要；点击展开系统工具。
- 左下：当前人物、职业、生活摘要与首要问题；点击展开人物中心。
- 右下：少量分级动态与聚合消息；点击展开动态历史。
- 同一层只保留一个主要中心。再次点击当前角落入口收起；打开新角落会替换旧中心。

## 左键、悬停与卡片

左键是所有主要操作。地图国家地区、城市、人物卡、组织卡和机构卡均以单击打开。悬停只显示紧凑来源、阈值、限制或接触渠道；单击才显示完整已知详情。内部 ID、原始字段和公式不进入普通原型 UI。

人物关系使用有限人物卡，不提供全人物下拉框。已有关系卡直接提供联系入口。我的组织和探索组织是人物中心中的两个独立视图，数据集合与主要动作不同。

## 关闭层级

`Esc` 依次关闭：人物详情或行动详情 → 地图对象卡 → 主要中心。无上层界面时不会退出。点击角落入口也可收起对应中心。

## 地图拖动、缩放与选择

- 在地图空白区域按住左键拖动平移。
- 鼠标滚轮缩放，缩放受静态配置上下限约束。
- 鼠标位于 UI 卡片或按钮时不缩放地图。
- 拖动超过阈值后不触发对象选择。
- 点击地区或城市更新对象卡和高亮；点击空白处取消选择。
- 法理归属、区域市场、人口和战争示例只改变静态视觉。

和平模式不绘制前线；战争示例模式才绘制静态假前线，并明确不运行战争模拟。

## 身份与权限

顶部示例身份切换器只服务原型演示。普通工人看到个人、住户、工作、工会、公开国家信息与新闻；地方官员额外看到所属机构、地方预算摘要、议程、辖区权限和可提交事务。完全无关的中央财政、外交和全国军事控制入口隐藏；已知但不满足条件的内容显示锁与一条原因。

## 动画

主要中心使用 220 毫秒四次 ease-out 打开、180 毫秒反向关闭，无弹跳。动效期间目标位置保持一致；遮挡区域由上层卡片先命中，避免点击穿透。

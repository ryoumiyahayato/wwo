# V2.1.2 静态原型交互规范

## 地图优先与四角入口

- 左上：当前国家或所属机构入口。
- 右上：日期、静态预览速度与展开箭头组成紧凑时间入口；速度菜单只含暂停和 1×/2×/4×/8×，独立齿轮菜单承载系统占位入口。
- 左下：当前人物的紧凑角标；人物中心打开时该角标不再绘制。
- 右下：最近一条重要通知、未读数和展开入口；不是常驻历史文本框。
- 下方中央：单个当前地图模式入口；点击才展开四种覆盖层。

国家、人物和动态主面板互斥。国家、人物与动态侧栏宽约 396 px；紧凑时间菜单与系统菜单独立逐层关闭。1280×720 下左侧面板不超过 420 px。

## 对象驱动交互

主要操作从地图国家、地区、城市、港口、机构、组织和关系人物进入。不存在独立“大行动中心”。对象卡第一层最多提供：

- 一个主要动作；
- 一个文字次要动作；
- 一个“更多”入口。

人物第一层是“联系 / 请求帮助 / 更多”；更多菜单提供引荐、调查和查看共同关系。关系详情直接显示完整结构与来源，不向玩家暴露界面层级术语。

## 四层信息密度

1. 扫视：对象名称、层级、一个关键状态和有限动作。
2. 悬停：来源、权限、线型或限制的一句说明。
3. 点击详情：完整已知结构、因果来源与上下文动作。
4. 隐藏 QA：内部 ID、原始字段与公式不进入普通画面。

这一规则同时适用于人物、组织、机构、地区、动态和行动。

## 身份结构

普通工人人物中心以生活、收支、工作合同、疲劳、关系、雇主、工会和个人计划为主。国家入口只展示公开政策、公开新闻和可接触组织。

人物姓名同时显示中文规范音译与法语原名。职业、雇主、工会、工会职位、行政机构和机构职位使用独立字段与独立行，不拼接为一条身份字符串。

地方官员除个人信息外，机构中心使用组织层级图，明确所属部门、上级、协作单位、管辖地区、预算来源、当前议程、可执行程序，并用“需要省长授权”“需要跨部门会签”“需要中央部门批准”等具体原因说明限制。两者不是同一页面增删按钮。

身份切换只属于 `--prototype-review` 评审工具；本地评审也可用 `F9` 临时显隐。普通运行和普通截图中不显示。

## 地图输入与缩放语义

- 地图空白处左键拖动平移；滚轮围绕指针缩放。
- UI 命中时不平移、不缩放；拖动超过阈值后不触发对象选择。
- 远景显示国家、国家名、少量主要城市和海洋。
- 欧洲中景显示当前国家、邻近主要国家、主要城市和至多四条主干铁路；隐藏法国内部宏观地区与地方节点。
- 法国近景在国家边界之上绘制九个由行政单位合并的宏观地区，再绘制自然行政边界、城市与交通；未选择对象时继续隐藏机构、组织和一般陆路。
- 选择城市或宏观地区后，才显示与该上下文相关的机构、组织和一般陆路；选择对象标签始终先于低优先级标签占用碰撞空间。
- 标签按优先级与碰撞矩形裁剪，不随缩放无条件放大同一批文字。

法理、市场、人口和战争状态共用 `natural_earth_110m_v2_1` 底图，只更换覆盖层。和平状态不绘制战线；战争示例仅增加静态控制覆盖、战线与战区标签。

当前世界政治边界与法国省级简化占位边界采用现代 Natural Earth 几何，仅用于 UI、缩放、标签和地图模式验证。国家名称及颜色包含 1900 年主题占位，不代表准确的 1900 年疆域、殖民体系或国际关系。

## 交通语义

- 铁路：深色双线、金色芯线和横向枕木，只连接陆地城市。
- 一般陆路：细短虚线。
- 航运：蓝色长虚线，连接港口并可跨海。
- 国境：浅色细实线。
- 战线：红色粗线与半透明控制覆盖。

## 关闭层级与动画

`Esc` 依次关闭模式菜单、人物/行动详情、地图对象卡和主要中心。无上层界面时不会退出原型。主要中心使用约 200 ms 四次 ease-out 打开、约 160 ms 关闭，无弹跳和点击穿透。

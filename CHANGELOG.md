# 更新日志

所有显著改动记录于此。版本发布前使用 `Unreleased` 段落。

## Unreleased

### Changed

- 完成 Codex 审计 P0/P1/P2 修复闭环：任意技能学习、实践成长、1000 种子保证成功、权威继承原因、NPC 旧区间结算、存档运行时不变量和 1280×720 固定开始按钮均取得绿色回归。
- 统一验证器改用独立进程捕获 Godot stdout/stderr，兼容 Windows PowerShell 5.1，校验精确引擎版本并扫描退出码为 0 时的脚本解析/加载错误；历史验证脚本改为转发到唯一入口。
- 历史综合测试夹具迁移到当前权威规则：领域服务负责地图写回、开发 UI 要求开发模式、退休夹具满足年龄条件、进行中行动存档通过真实入口创建。
- 修复继承事务回滚在运行时活跃上限变化时无法恢复名册的问题；外部存档仍严格执行配置上限，内部事务快照可恢复继承前状态。
- 玩家旅程测试收敛到唯一权威实现，旧命令与 post-audit 命令保留为兼容入口，消除旧鼠标滚动模拟器的警告和假失败。
- 人工可玩性复审后重新打开 P0 核心闭环，新增 `P0-R1` 修复路线，不再把底层服务测试等同于普通玩家可玩性完成。
- 战略地图顶栏改为可换行布局；玩家入口改为明确的“人物信息 / 长期行动 / 社会系统 / 保存游戏 / 开发工具”，用于修复 1280×720 下玩家功能可能被挤出窗口的问题。
- 长期行动不再允许玩家任意填写准备、资金、组织支持、关系支持和目标阻力；这些输入现在从人物财富、技能、情报、真实关系、组织职位和目标状态自动计算，行动开始时扣除实际费用。
- 社会系统中的直接加入组织、指定职位、直接建立关系和人物层级调整改为开发模式专用；普通玩家必须通过长期行动改变正式状态。
- 人物信息页面的隐藏资质与倾向事件入口改为开发模式专用。
- 地图直接施加压力和模拟控制易手入口改为开发模式专用，并在处理函数内部再次校验开发模式，隐藏控件不再等同于权限控制。
- 权威时钟、地图服务、社会服务、当前行动和自动存档协调器改为跨 UI 场景复用，避免查看人物后重建世界。
- 活跃 NPC 增加最小执行层：日常休息、工作、学习、加入组织与维持关系；具备权限的组织领导可在月度结算中产生地区社会影响或军事压力。
- 加载入口会先解析和验证存档；恢复失败会回滚地图与时钟、清理无效会话并返回主菜单显示错误，不再停留在无人物地图。
- 存档恢复改为先构建并验证临时人物、组织、关系、AI、行动、日志与性能状态，再提交权威地图、时钟和会话；失败路径保持原世界不变。
- 已完成行动的领域结果无论产生变化与否都只消费一次，避免满槽位、已达上限或无领域钩子的行动在刷新时反复尝试。
- 玩家旅程测试统一通过可见、启用且位于视口内的正式控件触发；控件不可用时立即失败，不再直接绕过禁用状态继续测试。

### Added

- 添加普通玩家可见的“保存游戏”按钮和明确的保存成功/失败反馈。
- 添加 `PlayerActionContextService`，集中从权威状态构建行动条件并扣除行动费用。
- 添加独立的 `p0_r1_safety_regression.gd`，覆盖存档事务回滚、不兼容配置拒绝和损坏主档的备份恢复。
- 初始化 Godot 4.6.3 Compatibility 项目。
- 添加基础中文主菜单、集中 UI 字符串和退出操作。
- 添加分级日志服务与无第三方依赖的 headless 测试运行器。
- 建立项目目录、核心文档、Git 忽略规则和 Windows 导出预设。
- 添加配置化权威游戏时钟，支持 1900 起始时间、暂停、1/2/4/8 倍速和单步推进。
- 添加小时、日、周、月信号以及按到期小时和插入顺序稳定处理的事件队列。
- 添加 M1 时间控制界面、菜单入口和帧率无关、日历边界及队列测试。
- 添加 `类型:slug` 稳定 ID 服务及可恢复的命名空间计数器。
- 添加国家、地区、控制单元、人口群体、人物、组织、关系和行动定义强类型模型。
- 添加统一确定性随机服务，支持固定种子、状态恢复、选取与洗牌。
- 添加两阶段核心数据加载器，验证字段、全局 ID 唯一性和跨实体引用并集中报告错误。
- 添加含 2 个架空国家、8 个地区、80 个控制单元、8 个人口群体和 4 座城市的正式世界数据。
- 添加程序绘制战略地图，支持拖动、缩放、实际控制色、法理/地区边界、争夺纹理、城市和铁路。
- 添加配置化控制压力、五阶段控制状态、控制易手及增量前线边集合。
- 添加地区/控制单元面板，以及边缘、孤立、包围、前线生成移除和地图交互测试。
- 添加 M4 配置化人物生成，支持明确国家、标准随机、全人口随机和按类别随机。
- 添加隐藏潜质、可见技能、分年龄性格表现、动态倾向、当前状态与困难开局标记。
- 添加人物创建和人物信息界面；正式视图隔离隐藏字段，开发者开关可检查精确生成数据。
- 添加八类配置化长期行动、行动实例和按权威时间差结算的进度服务。
- 添加技能、权限、组织/关系支持、准备、资金、隐藏资质、人物状态和目标阻力的集中公式。
- 添加暂停、恢复、取消、状态中断、六档定性把握、普通/必然成功与一次性结果应用。
- 在战略地图加入行动面板，并让地区控制支援结果连接现有控制服务。
- 添加政府、军队、企业和工会共 8 个正式组织，以及三级职位、槽位和权限结构。
